## Todos los Proyectos

### Proyecto 1: Plataforma SaaS de Gestión de Proyectos

**Stack tecnológico:**

- Next.js, React, Node.js, Express, PostgreSQL, Prisma ORM, JWT, TailwindCSS

**Funcionalidades:**

- Registro e inicio de sesión
- Creación de equipos
- Tableros de proyectos
- Listas de tareas
- Drag and drop de tareas
- Comentarios en tareas
- Sistema de roles (admin/miembro)
- Notificaciones

**Requisitos:**

- Arquitectura limpia
- API REST
- Validación de datos
- Manejo de errores
- Estructura profesional de carpetas
- Uso de TypeScript
- Documentación del proyecto
