# Portafolio Backend: Carlos Andres Martinez

## Minimalist Backend Portfolio
API para portafolio personal de desarrollador backend nivel medio.

## Stack
- TypeScript + Node.js
- Express + MongoDB
- JWT + Swagger

## Features
- Autenticación JWT
- CRUD usuarios/proyectos/habilidades
- Documentación API automática
- Seguridad integrada

## Arquitectura
```
src/
├── config/
├── controllers/
├── middleware/
├── models/
├── routes/
├── services/
├── utils/
├── validators/
└── app.ts
```
