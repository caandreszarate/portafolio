# Actualización de Proyectos

## Nuevos Proyectos Agregados

### 1. Plataforma SaaS de Gestión de Proyectos

**Stack tecnológico:**

- Next.js
- React
- Node.js
- Express
- PostgreSQL
- Prisma ORM
- JWT authentication
- TailwindCSS

**Funcionalidades:**

1. Registro e inicio de sesión
2. Creación de equipos
3. Tableros de proyectos
4. Listas de tareas
5. Drag and drop de tareas
6. Comentarios en tareas
7. Sistema de roles (admin / miembro)
8. Notificaciones

**Requisitos técnicos:**

- Arquitectura limpia
- API REST
- Validación de datos
- Manejo de errores
- Estructura profesional de carpetas
- Uso de TypeScript
- Documentación del proyecto
